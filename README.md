\# Job Alert Bot



A Python automation tool that fetches remote job listings from an API, filters jobs by keywords like Python and AI, removes duplicate results, and saves matching jobs to a local text file.



\## Features



\- Fetches real remote jobs from RemoteOK API

\- Searches across multiple job fields

\- Filters jobs by keywords

\- Prevents duplicate saved results

\- Saves matching jobs locally



\## Technologies Used



\- Python

\- Requests

\- JSON APIs

\- File Handling

\- Deduplication Logic



\## Workflow



1\. Fetch job listings from RemoteOK

2\. Loop through job data

3\. Search for keywords such as Python or AI

4\. Save only matching jobs

5\. Avoid saving duplicates
